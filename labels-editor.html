How to get your API key

If you use Claude Code CLI, your key is already stored locally:

  • Open %USERPROFILE%\.claude\config.json
  • Copy the value of primaryApiKey

Otherwise, generate one from the Anthropic console:

  1. Go to console.anthropic.com and sign in
  2. Click Settings (gear icon, bottom-left) → API Keys tab
  3. Click Create Key, give it a name, then copy it
🔒 Privacy: Your API key is saved exclusively in your browser's localStorage and is only ever sent directly to api.anthropic.com to perform translations. It is never shared, logged, or used for any purpose outside this tool.
